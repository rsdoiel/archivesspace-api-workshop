4. Repositories

Creating our create_repo function

The elements we change with each new create request will form the parameters in our function. Two elements are required and the rest are optional.

required

  • name
  • repo_code (needs to be unique)

optional

  • org_code
  • image_url
  • url

What we also need

  • We will need to submit our access token and provide the api url too
