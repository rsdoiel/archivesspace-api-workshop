5. Working with Agents

create_agent concerns

The API docs we want to read closely are the models/schema for the specific agent type we want to handle. The API will validate our fields so we’ll need to handle the response case where the API is trying to tell us we’re missing a required field or something else is out of order.

This has the benefit of keep our code simple but does mean we need more code evaluating the response.
