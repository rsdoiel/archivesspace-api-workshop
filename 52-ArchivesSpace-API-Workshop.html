5. Working with Agents

Our recipe

Each ArchivesSpace API Model tends to have its own nuances based on the schema. Agents is no exception. Rely on the docs as much as you can. When the docs fail there is the email list and you can experiment too. When I first started working with AS, its documentation was thin and I used curl and jq.

  1. Look up in the API Docs the object module that applies
    • Is there more than one?
  2. Look up in the API Docs the activity want to use
  3. Look at the HTTP method required (e.g. GET, POST, DELETE)
  4. Look at any data that needs to be sent, noting how it is sent
    • (e.g. a JSON blob or as part of a path)
